# Tourism Management System

## Install Dependencies
- JDK (latest)
- IDE (Netbeans)

Required:
1. JDK
2. MySQL Connector Java
3. JDBC
4. Eclipse IDE
5. XAMPP
## Database :
Create a database tourism in xampp and got ``./database/tourism.sql`` and drag and drop the file in it.
